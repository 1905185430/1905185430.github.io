# My Projects

Here are the main robotics and AI projects I have worked on. Each project includes detailed documentation, code, and results.

## Robot Learning & Control

### [SO-101 Robot Arm System](so101)
A comprehensive system for SO-101 dual-arm robot control, data collection, and model deployment. Features teleoperation, calibration, and policy training.

**Technologies**: Python, LeRobot, PyTorch, OpenCV  
**Status**: Active Development  
**GitHub**: [so101](https://github.com/1905185430/so101)

### [LeRobot Framework](lerobot)
Contributions to HuggingFace's LeRobot framework for robot learning. Includes dataset tools, policy training, and deployment utilities.

**Technologies**: Python, PyTorch, HuggingFace  
**Status**: Active Development  
**GitHub**: [lerobot_workdocs](https://github.com/1905185430/lerobot_workdocs)

### [VLA Model Fine-tuning](vla-finetune)
Tools and pipelines for fine-tuning Vision-Language-Action models like SmolVLA for specific robotic tasks.

**Technologies**: Python, PyTorch, Transformers  
**Status**: Active Development  
**GitHub**: [vla-finetune](https://github.com/1905185430/vla-finetune)
