# Jiaxuan Wang

**Robotics Engineer & Researcher**

Welcome to my personal website! I am a passionate robotics researcher focusing on robot learning, vision-language-action models, and computer vision. Currently, I am working on projects involving SO-101 robot arms, LeRobot framework, and VLA model fine-tuning.

## Research Interests

- **Robot Learning**: Imitation learning, reinforcement learning for robotic manipulation
- **Vision-Language-Action Models**: Fine-tuning and deploying VLA models for robot control
- **Computer Vision**: 3D perception, depth cameras, and visual servoing
- **Robot Control**: Teleoperation, calibration, and real-time control systems

## Current Projects

### SO-101 Robot Arm System
A comprehensive system for SO-101 dual-arm robot control, data collection, and model deployment.

### LeRobot Framework
Contributing to HuggingFace's LeRobot framework for robot learning with standardized datasets and policies.

### VLA Model Fine-tuning
Developing tools for fine-tuning Vision-Language-Action models like SmolVLA for specific robotic tasks.
