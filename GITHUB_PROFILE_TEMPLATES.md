## 📝 使用说明

### 1. 复制模板
选择一个你喜欢的模板，复制内容到你的GitHub个人资料README中。

### 2. 修改内容
将模板中的占位符替换为你自己的信息：
- `[Your Name]` → 你的名字
- `[Your Title]` → 你的职位
- `username` → 你的GitHub用户名
- `[Project Name]` → 你的项目名称

### 3. 添加统计
将统计卡片的URL中的`username`替换为你的GitHub用户名。

### 4. 自定义主题
修改统计卡片URL中的`theme`参数来改变主题：
- `radical` - 激进主题
- `dark` - 暗色主题
- `tokyonight` - 东京之夜主题
- `matrix` - 黑客帝国主题

### 5. 本地测试
在本地Markdown编辑器中预览效果，确保所有链接和图片正常显示。

## 🎯 最佳实践

### ✅ 推荐做法
- 保持简洁，不要过度装饰
- 定期更新内容
- 使用高质量的图片和图标
- 确保所有链接有效
- 添加个人特色

### ❌ 避免事项
- 不要使用过多的动画效果
- 不要添加不相关的徽章
- 不要复制他人的内容而不注明来源
- 不要使用过大的图片
- 不要添加敏感信息
